Componentes

Arduino Uno

Placa que permite la creación de una programación electrónica gracias a su plataforma de código abierto. Incluye el cable USB.

Módulo lector RFID

Lector y grabador de 13.56 MHz que permite trabajar de manera sencilla con los microcontroladores gracias a su sistema SPI. Para comprobar el lector se puede utilizar una tarjeta TAG o un llavero TAG.

Protoboard

Herramienta que permite realizar conexiones entre componentes electrónicos sin necesidad de realizar soldaduras.

Pantalla LCD

Conocida también como pantalla de cristal líquido, utiliza cantidades pequeñas de energía eléctrica y mide 7 x 2.5 cm.

Micro servomotor SG90

Dispositivo que permite controlar con máxima precisión la posición y movimientos de su eje. Se puede monitorear velocidad y ángulo.

Cable Dupont

Conocido también como Jumper, permite realizar conexiones desde cualquier cabezal macho o hembra en cualquier placa.

Diodo LED

Permite mostrar de manera sencilla cuando hay paso de energía eléctrica. En este caso indicaremos un mensaje con la luz.

Resistencia 220 Ω

Componente semiconductor cuya principal función es limitar el paso de corriente eléctrica en un circuito, en este caso al diodo.

Nuestra solución es crear un juego de baúles que les permita estar en movimiento, conocer tecnología actual (que les servirá para tener una noción si es que instalan esta tecnología en puertas del asilo) y ejercitar tanto músculos como neuronas porque va a ser un juego físico como mental. El equipo lo propone para combatir esta falta de actividad física o sedentarismo por parte de los adultos mayores.

En este apartado explicaremos el funcionamiento del proyecto mediante un diagrama de flujos.

¿En qué consiste el juego?

Principalmente se basa en un juego de destreza mental y que la "llave" que cada paticipante escoga sea la correcta con el baúl que cada uno escoga. Cada baúl tendrá un módulo lector RFID que podrá leer la señal solamente de una llave, si la llave es correcta se encenderá un diodo LED y aparecerá una palabra en la pantalla LCD relacionada a una pregunta que tenga el moderador del juego. Si cada participante responde de manera correcta, entonces se hará acreedor de otra "llave" que le permitirá tener más opciones para abrir baúles y así encontrar el que tenga el premio. Cabe mencionar que solo un baúl contiene este premio. Siempre tiene que haber una baúl más por la cantidad total de participantes y se eliminan de acuerdo a la cantidad de participantes eliminados en cada ronda. Este proceso es un bucle hasta que haya un ganador.

Desarrollo del proyecto en la plataforma Arduino IDE

¿Cuál es el desarrollo de manera precisa el proyecto RFID?

Se inicia con la conexión del módulo lector RFID, la pantalla LCD con el módulo I2C y el Micro servomotor SG90 al arduino y protoboard mediante cables Dupont macho-hembra. Es importante probar cada componente por separado con su respectivo código para verificar su correcto funcionamiento. Para esto hay que descargar la librería MFRC522, LiquidCrystal_I2C y Servo para que el código pueda funcionar de manera óptima. Si todos los componentes funcionan de manera adecuada, se pasa a la creación de un código global que enlace los tres componentes más la función de enceder el diodo adecuado para cada mensaje.

    Comprobación del circuito electrónico

    En este programa se pudo desarrollar el circuito en base a una pantalla LCD 16x2 con dos botones y diodos LED de colores rojo y verde para indicar la concesión o negación del acceso, y el servomotor SG90 para simular la apertura de una cerradura. Tanto con el color como en el mensaje y el giro del servomotor le indica a la persona que acciona este sistema si es que al apretar el botón se le está concediendo o no el acceso.

    En la ventana inferior se puede apreciar mediante un video el funcionamiento del circuito de reconocimiento. Si el botón es el indicado para aperturar el sistema, el ángulo del micro servomotor pasará de 0 a 90 grados y luego regresará al punto de origen, que en este caso será 0 grados. Si el botón no es el indicado, la acción del micro servomotor será nula.

    El primer paso es activar el sistema para la simulación. Lo siguiente sería apretar un botón para accionar la pantalla.

    Si se presiona el botón derecho, se activará el diodo rojo y el micro servomotor mantendrá su posición.

    Si se presiona el botón izquierdo, se activará el diodo verde y el micro servomotor cambiará su posición, de 0 pasará a 90 grados y volverá a su posición original.

El surgimiento de la primera forma del prototipo se debe al hecho de buscar un objeto que contenga una puerta para poder aplicar el sistema RFID y que a su vez se amolde al concepto de juego. Apareció la forma de un baúl por las dos características mencionadas y porque es uno de los contenedores que posee mayor historia tanto en lo narrativo como en lo cotidiano.

Es momento de diseñar la cara del proyecto

El proceso se inició en la clase presencial cuando el docente nos dio una indicación, previa a la clase, de llevar cualquier material que nos pueda ayudar a elaborar una figura en tres dimensiones. Cada equipo debía elaborar seis propuestas de un posible baúl que contendría todo el sistema que íbamos a explicar. Cada integrante desarrolló un prototipo, a base de cartón micro corrugado, de acuerdo a la investigación que encontró en base a su forma, tamaño u otras características y estas alentasen al usuario a querer participar de la dinámica que se propuso.

Antes de cortar piezas y pegarlas, se realiza el proceso más importante para un artista que es el de dibujar a mano alzada o con instrumentos. Esto es fundamental para realizar distintos bocetos en una hoja y así poder mejorarlos o combinarlos de acuerdo a la creatividad de cada uno. Cinco de las seis propuestas fueron representar básicamente un contenedor clásico o la forma icónica del baúl con terminación curva, pero aún así sirvió para entender cómo se sentiría cada anciano que viera esta propuesta desde el comentario de cada miembro con relación a cada baúl. El diseño faltante se acercó más a la idea de simular un rompecabezas debido a su forma trapezoidal que permitía que encajaran, logrando un ahorro de espacio al momento de almacenar estos. Aunque esta idea surgió más desde los conceptos de interiorismo que de psicología.

Finalmente cada miembro tuvo que exponer frente a la clase en menos de un minuto y debidamente sustentado el diseño que presentaba, quizás como una prueba para la exposición final.

    ¿Cómo fue el proceso?

    En la segunda clase dedicada al desarrollo de la maqueta, se escogió un diseño de los doce propuestos (ver modelos líneas abajo) en Tinkercad para elaborarlo de manera física con un ligero cambio de acuerdo a la perspectiva del equipo y a la solución que ya tenían desarrollada, que era el hecho de contar con un espacio interno para el sistema RFID y otro para colocar un premio. Cada miembro del equipo se encargó de una parte y con la ayuda del escalímetro se pudo adaptar las medidas de cada componente electrónico al diseño.

    En la imagen superior se puede observar el resultado de la maqueta de esa clase que representa la idea de contener un baúl con otro, pero de distinta forma. Se priorizó cambiar de material por cartón doble corrugado debido a su alta resistencia. En consecuencia a la falta de material es por lo que el tamaño es más pequeño. Finalmente el equipo acordó replantear la idea a escala real y el diseño con una coordinación previa durante el paso a paso del armado para su aprobación.

¿Cómo hacer sostenible el proyecto EnigmaBox?

Si bien el proyecto está pensando en ser un contenedor que pueda servir para almacenar un premio y que a su vez sea un juguete, pueda funcionar de manera correcta como un objeto que permita guardar elementos por bastante tiempo como si fuera un baúl o una maleta. Mencionado esto considero que podría solventarse como un elemento que facilite el almacenaje en cualquier lugar, pudiéndose adaptar a la medida que lo requiera cada cliente o espacio, ya que gracias a los conectores que posee en las caras laterales contribuye a un orden óptimo al momento de apilar estos baúles. Principalmente nuestro cliente mayor sería la Beneficencia de Lima debido a que se encargan de todos los centros de retiro en la ciudad de Lima y podrían abastecer de este contenedor a todos los centros debido a sus características beneficiosas. En segundo lugar, cualquier empresa que se encargue de desarrollar la tecnología RFID se podría asociar con el grupo EnigmaBox para apoyar con los componentes electrónicos y capacitaciones para las personas que estén interesadas en este sistema, llegándose a concretar si los resultados son positivos varias ventas o instalaciones. Un gran margen que beneficia a esta invención es que no se cierra solamente en adultos mayores, cualquier persona que estén interesada en un juego, por ejemplo niños, o quiera aprender el uso de este sistema de manera didáctica lo podría adquirir sin ningún problema e inclusive lo podría personalizar de acuerdo al color o medidas del baúl.

¿Qué tipo de tecnología complementaría a EnigmaBox?

¿Cuál es el desarrollo de manera precisa el proyecto RFID?

La más factible podría ser asociarla a la tecnología de Internet de las cosas (IoT) debido a que este contenedor puede servir como una caja fuerte para guardar elementos valiosos ya activarlo mediante una aplicación segura que incluya un código de seguridad y lectura de huellas digitales de la persona titular en caso se necesite abrir este baúl y no se cuenta a la mano con un llavero, tarjeta o pulsera TAG. Otra alternativa podría ser el uso de la realidad aumentada y que el baúl esté acondicionado para poder ser el recpetor de esta tecnología y se puedan observar distintos temas de manera gráfica o visual como complemento a la misma dinámica de la pregunta según el tópico que aparezca en la pantalla LCD. Esto ya tiene un precedente como lo es el ejemplo de Merge Cube.

  • ¿Es posible la escalabilidad en el proyecto EnigmaBox?

    Considerando que cuenta con el sistema básico de la tecnología RFID este proyecto, sí se podría generar una actualización de este sistema mediante la inclusón de una pulsera TAG para que sea más cómoda para los participantes al momento de jugar, además no habría riesgo de pérdida porque no hay la necesidad de retirarse la pulsera. También pueden variar según la frecuencia con la que cuente cada baúl ya que actualmente existen tres tipos: Ultra-Alta Frecuencia (UHF), Alta Frecuencia (HF) y Baja Frecuencia (LF). Eso es en cuestión de algo técnico, pero también habría la alternativa de enfocarse solo en lo estético como la aplicación de luces LED interiormente mediante una ranura para que pueda servir como elemento luminoso o una lámpara. Otra opción sería ponerle un nombre en alto relieve al baúl para personalizarlo.
