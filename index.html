More

Proyecto

Consiste en realizar una simulación de una apertura de una puerta mediante una tarjeta o llavero gracias al sistema RFID. Esto se lleva a cabo gracias a un módulo sensor que reconoce el elemento, previo registro de sus datos para su reconocimiento.
